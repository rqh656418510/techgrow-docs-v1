# 侵权投诉处理规则

本平台尊重并保护知识产权。

如权利人认为平台用户通过本平台服务侵犯其合法权益，可以按照以下方式提交侵权投诉。

## 一、投诉材料

投诉人应提供以下材料：

1. 权利人身份证明
2. 权利证明材料（如著作权证明）
3. 侵权内容具体链接
4. 侵权情况说明
5. 联系方式

投诉材料应保证真实、合法、有效。

## 二、投诉提交方式

请将投诉材料发送至平台指定邮箱：

`656418510@qq.com`

或通过平台投诉渠道提交。

## 三、处理流程

平台在收到完整投诉材料后，将按照以下流程处理：

1. 审核投诉材料
2. 通知相关用户
3. 根据情况采取必要措施

必要措施包括但不限于：

- 删除相关内容
- 屏蔽访问
- 限制用户账户

## 四、恶意投诉

如发现投诉人存在恶意投诉行为，平台有权拒绝处理相关投诉。

投诉人应对其投诉材料的真实性承担法律责任。

## 五、责任声明

本平台仅提供技术服务。

用户发布的内容及其法律责任由用户自行承担。
